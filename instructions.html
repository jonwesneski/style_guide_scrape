Style Guide Scraper

A tool to scrape images on a single page of a website. This tool has config settings to run this script, the main 2 settings are:

  • url: The http/https page/url you want to scrape
  • output-folder: Where do you want to save the images
*The other settings will be explained towards the botom of the page

Instructions

  1. Create the config:
    • Double click on the app "StyleGuideScraper" (This will create a config.ini file)
      *If at anytime this file gets corrupted, you may delete the config.ini file and then double click on the app again
  2. Set the config:
    1. Double click on the config.ini file to open it
    2. Change the url setting (change the text after the "=" sign)
    3. Change the output-folder setting (change the text after the "=" sign)
    4. Save the file
  3. Run the script and wait for it to finish:
    • Double click on the app "StyleGuideScraper"

Advanced Filter Settings

Here are the advanced settings to help filter down to what kind of images you are looking for:

  • image-url-sub-path: A common pattern is to put images into a certain path to group them together, for example:
    https://www.mycdn.com/shop/files/flagship.jpg
    https://www.mycdn.com/shop/files/redshirt.jpg
    Both of these are inside "shop/files" at "https://www.mycdn.com"
    If there are other images, for example: https://www.mycdn.com/common/files/mylogo.jpg
    It will not be downloaded

    *If you just want to grab all images you can set this setting to: /
  • min-image-size-kb: Images come in all dimensions and sizes, and a website may have several images that are not "style guide" images, like a logo, some custom UI thing, or even a much smaller thumbnail of what you actually want. To help filter these out you can see what the average image size and use that as a setting here

    *Also note that some sites might have bigger than expected image sizes, so you may have to increase the image size setting for those sites
  • file-extensions: There are many image file extensions out there: .jpg, .jpeg, .png are some of the common ones. To filter to the ones you want you can set this. Determining the file extension is looking at the url itself and seeing what the url path ends with a "dot something", for example:
    https://www.mycdn.com/shop/files/flagship.jpg - This one is .jpg
    https://www.mycdn.com/shop/files/flagship.png - This one is .png

    To grab just one type of file extension it is simply a .jpg or a .png or whatever the file extension is

    To grab more than one type, include a comma between the different types:
    .jpg,.png,.jpeg
